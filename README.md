# Maquinarias Landau - Landing Page

Landing page premium para Maquinarias Landau, distribuidores oficiales de Bosch y Makita con más de 75 años de trayectoria.

## Características

- Diseño moderno estilo Bosch con fondo blanco
- Sección Hero con animaciones
- 6 servicios con flip cards 3D interactivas
- Métricas animadas
- Testimonios de clientes
- Formulario de contacto
- Diseño responsive

## Tecnologías

- Next.js 16
- React
- TypeScript
- Tailwind CSS
- shadcn/ui

## Despliegue

El proyecto está configurado para desplegarse automáticamente en Vercel.

### Variables de entorno

Crea un archivo `.env` con las variables necesarias:

```env
# Base de datos (si se requiere)
DATABASE_URL="tu_database_url"
```

## Scripts

```bash
# Desarrollo
npm run dev

# Build de producción
npm run build

# Iniciar servidor de producción
npm run start
```
